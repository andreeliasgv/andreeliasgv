# CSS3

## O que é
CSS3 é a linguagem de estilo usada para controlar aparência e layout de páginas HTML.

## Por que aprender
- Cria interfaces atraentes e usáveis
- Suporte a layouts modernos (Flexbox, Grid)
- Transições, animações e responsividade

## O que estudar aqui
- Seletores e especificidade
- Box model (margin, border, padding)
- Flexbox e Grid
- Media queries para responsividade
- Variáveis CSS (custom properties) e boas práticas
- Transições e animações básicas

## Projetos sugeridos
- Clone visual de uma página (ex: page layout)
- Layout responsivo com Grid + Flexbox
- Pequenas animações de UI (hover, cards)

## Recursos / Materiais
- Snippets para layout (container, grid-template)
- Cheatsheet de Flexbox e Grid
- Padrões de design simples (cards, navbar)

## Exercícios rápidos
1. Fazer um layout de 3 colunas que vira 1 coluna em telas pequenas.
2. Criar um card com efeito hover usando transição.
