# HTML5

## O que é
HTML5 é a linguagem de marcação usada para estruturar conteúdo na web — textos, imagens, vídeos, formulários e mais.

## Por que aprender
- Base do desenvolvimento web
- Essencial para front-end
- Trabalha junto com CSS e JS

## O que estudar aqui
- Estrutura básica (doctype, head, body)
- Semântica (header, nav, main, article, section, footer)
- Formulários e validação
- Elementos multimídia (audio, video)
- Acessibilidade básica (aria, labels)

## Projetos sugeridos
- Página pessoal (portfolio)
- Landing page responsiva
- Formulário com validação HTML5

## Recursos / Materiais
- Boilerplate HTML
- Checklist de acessibilidade
- Lista de tags semânticas com exemplos

## Exercícios rápidos
1. Criar uma landing page com header, hero e formulário.
2. Marcar corretamente a hierarquia de títulos (`h1`..`h3`) para SEO.
