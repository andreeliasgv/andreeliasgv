# JavaScript

## O que é
JavaScript é a linguagem de programação do navegador (e também do servidor com Node.js). Permite tornar páginas interativas.

## Por que aprender
- Interatividade no front-end
- Ambiente unificado (full-stack com Node)
- Grande ecosistema (npm) e frameworks

## O que estudar aqui
- Sintaxe: variáveis, funções, escopo
- DOM: selecionar e manipular elementos
- Eventos e manipulação assíncrona (Promises, async/await)
- Módulos ES6, fetch/API
- Ferramentas: bundlers (Vite, webpack) e NPM
- Testes básicos (Jest)

## Projetos sugeridos
- To-do list dinâmica (localStorage)
- Pequena aplicação consumindo uma API pública
- App com rotas simples usando um router (ou SPA básica)

## Recursos / Materiais
- Boilerplate JS + exemplo de fetch
- Checklist para debugar JS (console, DevTools)

## Exercícios rápidos
1. Implementar um contador que incrementa/desincrementa no DOM.
2. Buscar dados de uma API e mostrar em uma lista.
