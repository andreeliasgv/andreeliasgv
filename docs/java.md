# Java

## O que é
Java é uma linguagem de programação orientada a objetos, com tipagem estática, amplamente usada para aplicações web, desktop, mobile (Android) e sistemas corporativos.

## Por que aprender
- Portabilidade (JVM)
- Grande ecossistema (bibliotecas, frameworks)
- Alta demanda no mercado
- Bom para backend, microservices e aplicações empresariais

## O que estudar aqui
- Sintaxe básica: tipos, controle de fluxo, classes e objetos
- Orientação a objetos: herança, polimorfismo, encapsulamento
- Coleções, Streams e generics
- Exceções e I/O
- Build tools: Maven / Gradle
- Testes: JUnit
- Frameworks: Spring Boot (introdução)

## Projetos sugeridos
- API REST simples com Spring Boot (CRUD)
- CLI para gerenciar tarefas (arquivo JSON/SQLite)
- Pequeno jogo (console) para treinar OOP
- Microserviço com persistência em banco (PostgreSQL)

## Recursos / Materiais
- Estrutura de pastas de exemplo
- Comandos Maven/Gradle básicos
- Checklist de boas práticas (naming, packaging)

## Exercícios rápidos
1. Implementar uma classe `ContaBancaria` com depósito/saque.
2. Criar API CRUD para "Notas" com Spring Boot.
